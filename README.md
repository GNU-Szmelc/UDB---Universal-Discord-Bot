# Universal-Discord-Bot
> A modular approach to Discord bots


## Modules & Commands:
> `shell.py`
```bash
shell <COMMAND>
```
> `cmd.py`
```bash
cmd <[figlet/toilet/cowsay/fortune/uptime]>
```
> `yt.py`
```bash
yt "search query"
```
> `dlp.py`
```bash
mp3 <URL>
mp4 <URL>
```
> `file_upload.py`
```bash
upload #channel /path/to/file
```
> `dm.py`
```bash
dm @user <Message>
```
> `tictactoe.py`
```bash
game @user1 @user2
```
> `role_manager.py`
```bash
listroles @user
addrole @user @role
removerole @user @role
```
> `rng.py`
```bash
coinflip
diceroll [6/20]
randomstring [Length]
```
> `losowe.py`
```bash
losowe
```
> `jokes.py`
```bash
joke
```
> `furry.py`
```bash
furry <search query>
```
> `emote.py`
```bash
textemoji
```
> `cycki.py`
```bash
cycki
```
> `crypto.py`
```bash
crypto
crypto <SYMBOL>
```
> `bomba.py`
```bash
bomba
```

```bash
boner
```
